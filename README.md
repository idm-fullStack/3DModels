  3D Mathematical Mesh Renderer

 "A 3D model can be made based on pure mathematics, as you can see in the project."

This repository demonstrates the rendering of a low-poly frog mascot using fundamental linear algebra, matrix transformations, and coordinate projections without relying on heavy external game engines.

 Mathematical Pipeline
* **3D Matrix Multiplication:** Displacement, scaling, and rotation (Pitch, Yaw, Roll) via trigonometric matrices.
* **Projection Matrix:** Converting 3D vector coordinates onto a 2D screen coordinate system ($X, Y$).

 Tech Stack
* **Language:** Python 
* **Core:** Matrix algebra, Vector calculus, Trigonometry.
